One-Time Setup

To save changes from the browser, paste a fine-grained GitHub token. It is stored only in this browser.

  1. Open github.com/settings/personal-access-tokens/new
  2. Token name: anything (e.g. morning-digest editor)
  3. Repository access: Only select repositories → morning-digest
  4. Repository permissions:
    • Contents: Read and write
  5. Click Generate token, copy it, paste below.
